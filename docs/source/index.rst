Bifrost a data analysis management system
===================================

Bifrost is in use as platform for data analysis management of bacterial whole genome sequence (WGS) samples. Right now the documentation is being updated and previous documentation is quite out of date. If you have questions please contact KIMN@ssi.dk until we get this updated.
